## What is Machine Learning?

Machine learning is basically teaching a computer to learn patterns from data instead of writing strict rules for everything.

For example, instead of manually defining what a spam email looks like, we can train a model using past emails and let it figure out the pattern on its own.

---

## Types of Machine Learning

### 1. Supervised Learning
This is when the model learns from labeled data (we already know the correct answers).

Examples:
- Predicting house prices (regression)
- Classifying emails as spam or not spam (classification)

### 2. Unsupervised Learning
Here, the model doesn’t get any labels. It tries to find patterns on its own.

Example:
- Grouping customers based on their behavior

### 3. Reinforcement Learning
This is more like trial and error. The model learns by getting rewards or penalties.

Example:
- Game AI or self-driving cars

---

## Regression vs Classification

This part was important to understand:

- **Regression** → output is a number (like price, salary, etc.)
- **Classification** → output is a category (like yes/no, spam/not spam)

---

## Neural Networks (Basic Idea)

Neural networks are inspired by how the human brain works.

They have:
- Input layer
- Hidden layers
- Output layer

The model learns by adjusting something called "weights" to improve its predictions over time.

---

## NLP (Natural Language Processing)

NLP is about helping computers understand human language.

Some common examples:
- Chatbots
- Translation (like Google Translate)
- Sentiment analysis (positive/negative text)
