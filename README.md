Todo List (Python OOP Project)

This is an interactive to-do list program built using object-oriented programming in Python. It allows users to add tasks, mark them as completed, and store them permanently using a JSON file.

Features
- Add tasks
- View tasks
- Mark tasks as completed
- Interactive menu system
- Save tasks in JSON file
- Load tasks automatically when program starts

Concepts Used
- Classes and objects
- Encapsulation
- Methods
- Lists
- File handling
- JSON
- Error handling

How to Run
Run the file using:
python main.py

Files
- main.py
- tasks.json
