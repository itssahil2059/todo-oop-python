Todo List (Python OOP Project)

This is a simple to-do list program built using object-oriented programming in Python. It allows users to add tasks, mark them as completed, and view all tasks.

Features
- Add tasks
- Mark tasks as completed
- View all tasks

Concepts Used
- Classes and objects
- Encapsulation
- Methods
- Lists

How to Run
Run the file using:
python main.py
